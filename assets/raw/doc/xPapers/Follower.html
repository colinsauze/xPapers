NAME

xPapers::Follower

DESCRIPTION

Inherits from: xPapers::Object

Table: followers

Registers that a user wants to follow an alias, i.e. wants to receive all articles signed by a particular name. Note that an author can use many variants of his name to sign articles - we call these variants 'aliases'.
