NAME

xPapers::OAI::Harvester

DESCRIPTION

Inherits from: Moose::Object.

The harvestRepo method retrieves data from the OAI repository saved in the repo attribute and creates xPapers::Entry objects out of it. The objects are then handled to the handle_entry method that does nothing in this class. The also included xPapers::OAI::Harvester::Acumulator overrides the handle_entry method and saves the entries into the array in the entries attribute. This can be used for checking out entries from an OAI repository.
