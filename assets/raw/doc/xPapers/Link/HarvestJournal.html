NAME

xPapers::Link::HarvestJournal

DESCRIPTION

Inherits from: xPapers::Object

Table: harvest_journals

This class represents journal sets from the OAI server at CrossRef.

The also included xPapers::Link::HarvestJournalMng class contains updateFromFile method that construct or updates the records according to the csv file normally found at http://www.crossref.org/titlelist/titleFile.csv.
