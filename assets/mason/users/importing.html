The bibliographic file importer

This tool reads bibliographic files (e.g. BibTeX files). Entries imported with this tool will be added to <% $s->{niceName} %> if they do not exist in the database already, as well as to a list / category if requested. One of the formats available is very similar to the APA notation. You can write and import files of this kind to save time if you need to add many entries to <% $s->{niceName} %> and don't have a ready-made list of them.
